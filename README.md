# Blog
An application used to read, write, update and delete blogs.
Built with Node, Express and MongoDB.

## Technologies

Project is created with:
NodeJS 14.5.4,
Express 4.17.1,
MongoDB

For testing:
Postman API

### Setup
To run this project install it localy using npm:

```
$ cd ./Blogs
$ npm install 
```
To Start Server:

```
$ nodemon
